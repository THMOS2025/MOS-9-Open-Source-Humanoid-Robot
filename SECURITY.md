# Security Policy

## Supported Scope

This repository is a coordination hub and contains metadata, documentation, and helper scripts.

Security-sensitive findings are most likely to belong to a specific sub-repository. Please report issues in the relevant project if possible.

## Reporting a Vulnerability

If you believe you have found a vulnerability related to this hub repository:

1. Do not open a public issue with exploit details.
2. Contact maintainers through a private channel if available.
3. Include reproduction steps, impact, and affected files.

We will acknowledge reports as quickly as possible and coordinate remediation.
